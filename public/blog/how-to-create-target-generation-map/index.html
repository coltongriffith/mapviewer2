How to Create a Target Generation Map

To create a target generation map, import your target polygon outlines as GeoJSON, assign the Target Areas role, add anomaly and structural layers, apply outside shading to focus attention on priority zones, label each target, and export at high resolution for technical presentations.

What You Need Before You Start

Gather the following before building your target generation map:

  • Target polygon outlines as GeoJSON (digitized from geophysics interpretation or structural mapping)
  • Anomaly extents as a separate GeoJSON layer (if available)
  • Fault and structural linework as GeoJSON
  • Property claims boundary GeoJSON
  • Company logo and project name

Step 1: Import Target Polygon Data

Open Exploration Maps and click the import button. Select your target outlines GeoJSON — each polygon should represent a discrete exploration target. The layer is added and the map fits to your data extent automatically. If you have multiple target tiers (e.g., Priority 1, Priority 2), import each as a separate layer so you can style them independently.

Step 2: Assign the Target Areas Role

Expand the layer card and select 'Target Areas' from the Role dropdown. This applies a warm orange stroke with a light fill — the conventional styling for exploration targets. For your highest-priority targets, switch the fill to a more saturated colour and increase the stroke weight to 2–3 px. For lower-priority targets, reduce opacity to 40–60% so the hierarchy is visually clear at a glance.

Step 3: Add Anomaly and Structural Layers

Import your geophysical anomaly extents and fault or structural linework as separate GeoJSON layers. Assign anomaly polygons a high-contrast fill (magenta or gold works well at low opacity) to communicate the extent of the geophysical signal. Assign fault lines a dashed dark stroke. Reducing the opacity on both to 50–70% keeps these layers readable without overwhelming the target polygons in the foreground.

Step 4: Apply Outside Shade to Focus Attention

In the Design section, enable the Outside Shade effect on your priority target layers. This darkens the map area outside the target polygon, drawing the viewer's eye directly to the area of interest. Use it selectively — applying it to too many layers simultaneously reduces the effect. Outside Shade is particularly effective when presenting a single high-priority target to investors or JV partners.

Step 5: Label Each Target

Click on each target polygon to open the inline feature editor. Enter the target name (e.g., 'Zone A', 'North Anomaly', 'Western Shear') as the label. For targets with associated geochemical peak values or historical resource estimates, add a brief subtext note. Labels anchor to the polygon centroid automatically — drag them to adjust positioning if polygons overlap.

Step 6: Export for Technical Presentations

Target generation maps are typically shown in technical investor presentations and JV partner packages, so the 16:9 Landscape ratio is standard. Select the Satellite basemap to provide terrain and vegetation context. Apply the Technical — Sharp Borders or Modern — Dark Indigo theme. Export at 3× resolution (PNG) for high-resolution display on large screens and in printed technical documents.

Frequently Asked Questions

What data format do I need for target polygon outlines?
GeoJSON is the required format. If your target outlines are in a GIS package (QGIS, ArcGIS, or MapInfo), export the layer as GeoJSON with CRS set to EPSG:4326 (WGS84). If you've sketched targets in a PDF interpretation, you'll need to digitize them in QGIS first — draw your polygon outlines, then export as GeoJSON.
How do I show multiple priority levels (Priority 1, Priority 2, Priority 3) on the same map?
Import each priority tier as a separate GeoJSON layer. Style Priority 1 targets with a fully saturated colour and 100% opacity. Apply 60% opacity to Priority 2 and 30–40% opacity to Priority 3. This creates a clear visual hierarchy that communicates target ranking without requiring a written legend explanation.
Can I show geophysical anomaly extents (magnetic, IP) alongside the target polygons?
Yes. Import your anomaly extents as a separate GeoJSON polygon layer. Use a semi-transparent fill (20–30% opacity) in a contrasting colour — gold or magenta works well against a satellite basemap. Place the anomaly layer below the target layer in the layer stack so the target outlines render on top.
What export resolution is needed for investor technical presentations?
Export at 3× (PNG) for technical presentations intended for display at full screen on a projector or large monitor. This produces a PNG at approximately 5760 × 3240 pixels for a 16:9 export, which is sufficient for any current display and prints cleanly to A3 or tabloid size.
Should I use a satellite or topographic basemap for a target generation map?
Satellite is the preferred basemap for target generation maps — it communicates terrain, vegetation cover, and outcrop distribution in a single layer, all of which are relevant to the geological story. Use Topographic if your property is in a heavily forested area where satellite imagery doesn't show ground features clearly, or if the structural linework is the primary focus and needs a clean background.
